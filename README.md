# Grafast Getting Started

Following through the getting started instructions in the Grafast documentation.

Look through [the commits](https://github.com/grafast/getting-started/commits/main/):

1. `Initial commit`: seed with a README
2. `Use Node 24+`: configure nvm to use Node v24
3. `Install yarn4`: install our preferred package manager
4. `Install prettier`: for code formatting
5. `Install graphql and grafast`: the runtime dependencies
6. `Add tsconfig.json`: configure TypeScript for Node v24+ type stripping
7. `Add schema and main`: create the [src/schema.ts](src/schema.ts) and
   [src/main.ts](src/main.ts) files
8. `Add codegen`: make `src/schema.ts` strongly typed via
   `graphql-codegen-grafast`
9. `Add add() step`: creates the `add()` step and replaces `lambda()` usage
10. `Summary`: produce this list of commits
